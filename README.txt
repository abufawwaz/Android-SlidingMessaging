Sliding Messaging Pro Version
-----------------------------

This is a private repo to work on Jacob and Luke's Sliding Messaging Pro android application.


## Dependencies
---

1) Modified libraries are in the Libraries/ folder, all of them need to be included
2) android-smsmms (https://github.com/klinker41/android-smsmms)
3) CardsUI (https://github.com/nadavfima/cardsui-for-android)
4) Drag sort listview (https://github.com/bauerca/drag-sort-listview)
5) Floating Notifications Extensions (https://code.google.com/p/fn-extensions-library/)
6) ListView Animations (https://github.com/nhaarman/ListViewAnimations)
7) Pager Sliding Tab Strip (https://github.com/astuetz/PagerSlidingTabStrip)
8) Sliding Menu (https://github.com/jfeinstein10/SlidingMenu)
